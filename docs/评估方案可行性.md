# PyML 转换为 QML 和 Python 代码时的问题

## 类实例传递

```
# pyml

class FSPath:
    _path: str


comp A:
    attr path: FSPath()
    on_completed:
        print(path)

```

将会发生什么?

```
// qml (A.qml)
Item {
    property var path: PyML.call('method_101419')
    // -> QVariant(PySide::PyObjectWrapper, )

    Component.onCompleted: {
        PyML.call('method_151804', root)
    }
}


# python
class FSPath:
    _path: str


class PyML(PyMLCore):

    def method_101419(self):
        return FSPath()

    def method_151804(self, root):
        prop0 = root.property('path')
        print(prop0)  # -> <__main__.FSPath object at 0x0000023251230FA0>

```
